# **App Name**: TaxWise Command

## Core Features:

- Tax Season Command Center: Dashboard for real-time operational monitoring of the tax preparation pipeline, tracking document status and compliance for all accounts.
- Digital Tax Control Sheet: Automated document attachment and classification for 1099s, K-1s, and other tax documents, reducing manual tracking and scanning errors.
- Incoming Tax Document Ingestion Queue: Automated document ingestion and matching using generative AI tool for processing and identifying tax documents from multiple broker portals, filtering and categorizing by document type and broker.
- LP/LLC Position Tracker: Tracking tool for LP/LLC and PTP positions, updating positions for tax reporting and distinguishing between PTP and private LPs.
- Distribution & Large Cash Review: Review distributions to clients and large cash flows, flagging suspicious transactions for tax reporting.
- Tax Worksheet Viewer: Consolidated tax worksheet resembling OneSource, providing a comprehensive tax summary from all brokers, sales data, and tax-exempt interest details.
- CPA Partner & Workload Overview: Manage external CPAs and track their workload, monitoring status and turnaround time.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and authority.
- Background color: Light gray (#F5F5F5), nearly white, for a clean and professional look.
- Accent color: Soft purple (#9575CD), a related hue, to highlight key actions.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use clean, modern icons to represent different tax documents and statuses.
- Use a clear, well-organized layout to present financial data effectively.
- Use subtle animations for loading states and transitions.
